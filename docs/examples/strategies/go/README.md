# Go strategy stub (unsupported)

**Not first-party.** dockly does not ship or support a Go strategy in v1
([ADR 0011](../../../adr/0011-dockly-product-vision.md)).

This directory sketches how a community Go strategy might look against the
Python `Strategy` contract (`matches` + `plan`). It is for contributors exploring
the API — not a runnable plugin.
